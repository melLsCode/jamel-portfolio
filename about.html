About Me

Building AI-Powered Mobile & Web Applications

I am a Flutter and React Native Developer specializing in Firebase, AI integration, and scalable mobile applications. I build intelligent mobile apps, web dashboards, and digital solutions that combine modern user experiences with reliable backend systems.

My experience includes developing AI-powered platforms, real-time analytics dashboards, and business-focused applications using Flutter, React Native, Firebase, OpenAI API, and Gemini API. I focus on creating clean, maintainable, and user-centered products that solve real-world problems.

Some of my notable projects include ReasonUp AI, an AI-powered critical thinking platform with rubric-based evaluation and analytics, EstateWise AI, a real estate investment assistant, VoxFil AI Voice Generator, and custom admin dashboards built with Flutter Web and Firebase.
